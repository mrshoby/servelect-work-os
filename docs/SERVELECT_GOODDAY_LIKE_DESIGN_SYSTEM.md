# SERVELECT GoodDay-like Design System — v8.1.0 update

## Direction
Servelect identity remains: dark/green enterprise accent, light content area, compact professional Work OS layout.

## Components refined in v8.1.0
- Evidence cards: version, status, readiness score, missing runtime.
- Status badges: ready/passed green, dry_run/queued amber, blocked red, neutral slate.
- Dense tables: mutation queue with entity/mode/ACL/provider/reconciliation.
- Admin panels: two-column layout with primary table + proof/right context.
- Production pilot pages: should look like final product control screens, not marketing dashboards.

## Do not do
- Do not copy GoodDay logo/assets/text.
- Do not add decorative redesigns before backend/persistence and critical Work OS functions are stable.
